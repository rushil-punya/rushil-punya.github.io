Shopper Alert

My team and I developed Shopper Alert for LA Hacks. Shopper Alert allows users to sign up for a text notifcation service and be alerted when items, which are typically low due to COVID-19, are in stock. This app uses a Twilio/Python integration along with Tkinter for the GUI.
GitHub and more details:

https://github.com/rushil-punya/ShopperAlert

https://buildforcovid19.io/shopperalert/

CAPSULE RUSH

For this project I used the Unity3D game engine along with C#.
Capsule Rush is a game in which the player has to aim and shoot a gun to defend themself from capsule-shaped enemies which progressively get faster and multiply in numbers. The plane which the player navigates on is a 3D scanned version of my desk with a tablecloth and various objects on it as obstacles.
